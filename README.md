# Nobel Prize Data Analysis (1901 - 2023)
An exploratory data analysis of Nobel Prize winners (1901-2023) using Python and Pandas.
### Project Overview
This project explores a century of Nobel Prize data to identify trends in gender, nationality, and age.

### Key Questions Answered:
* Which country has the most Nobel laureates?
* When did the USA begin its dominance in the awards?
* Which categories have the highest proportion of female winners?
* Who are the elite few who have won multiple prizes?

### Tools Used
* **Python** (Pandas for data manipulation)
* **Seaborn/Matplotlib** (Data visualization)
